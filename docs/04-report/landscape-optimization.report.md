# [Report] 가로 모드 및 대화면 최적화 (v3.8.0)

## 1. 프로젝트 개요
- **목표**: 태블릿 가로 모드 및 대형 모니터 사용자들을 위해 화면 공간을 최대한 효율적으로 활용하는 'Split-view' 및 'Dual-pane' 아키텍처를 도입함.
- **결과**: v3.8.0 업데이트를 통해 대화면 전용 2분할 레이아웃, 확장된 대시보드 그리드, 고밀도 시간표 구현 완료.

## 2. 구현 내용
### 2.1 사이드바 및 전역 레이아웃 (v3.8.0)
- **상시 노출 사이드바**: `lg` 해상도 이상에서 네비게이션 사이드바를 왼쪽에 고정하여 메뉴 접근성을 높임.
- **컨텐츠 영역 확장**: `xl` 이상의 해상도에서 컨텐츠 최대 너비를 `1600px`로 확장하여 광활한 모니터 공간 활용.

### 2.2 2분할 참조 레이아웃 (Dual-pane)
- **7:3 화면 분할**: 학급 일지, 출결 관리, 누가 기록 화면에서 메인 입력 폼과 참조 위젯을 동시에 표시.
- **Sticky Reference Panel**: 스크롤 시에도 오늘의 날씨, 분위기, 출결 요약, 미니 시간표 등이 상단에 고정되어 기록 중 정보를 즉각 참조 가능.

### 2.3 대화면 최적화 위젯
- **3열 대시보드**: `xl` 이상 해상도에서 차트, 요약, 할 일을 한 줄에 배치하여 대시보드 가독성 극대화.
- **확장형 시간표**: 대화면에서 수평 스크롤을 제거하고, 셀 크기와 폰트 사이즈를 상향하여 주간 전체 일정을 한눈에 파악.
