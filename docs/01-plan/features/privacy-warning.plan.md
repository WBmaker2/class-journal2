# [Plan] 학생 개인정보 보호 알림 (privacy-warning)

## 1. 개요
- **기능명**: 학생 개인정보 보호 알림 모달 (Privacy Warning Modal)
- **목표**: 교사가 학급일지나 학생 누가기록을 PDF/Excel로 내보낼 때, 민감한 개인정보가 포함되어 있음을 인지시키고 안전한 보관 및 파기를 유도함.
- **주요 기능**:
    1. **경고 모달 표시**: 다운로드 전 경고 모달 창 띄우기.
    2. **하루 동안 보지 않기**: '오늘 하루 이 알림창 보지 않기' 옵션 제공.
    3. **수락 후 다운로드**: 사용자가 안내를 확인하고 '확인'을 누를 때만 내보내기 진행.

## 2. 사용자 스토리
- "교사로서 학생의 민감한 데이터(학급일지, 누가기록 등)를 로컬로 다운로드할 때, 보안에 대한 경각심을 가질 수 있도록 안내 문구가 있었으면 좋겠어."
- "매번 다운로드할 때마다 경고창이 뜨면 번거로우니, 오늘 하루 동안은 다시 보지 않게 설정할 수 있으면 좋겠어."

## 3. 상세 요구사항
### 3.1 알림 모달 UI
- 시각적으로 경고의 의미를 담아 빨간색 계열의 UI 요소 사용.
- 안내 내용: 암호 설정 보관, 외부 유출 금지, 사용 후 즉시 영구 삭제 등.

### 3.2 오늘 하루 보지 않기 기능
- 체크박스를 통해 상태를 입력받음.
- LocalStorage를 활용하여 `hidePrivacyWarning_YYYY-MM-DD` 키로 상태 저장.
- 저장된 키가 유효한 날짜(오늘)와 일치하면 모달을 생략하고 즉시 다운로드 실행.

## 4. 제약 사항 및 고려 사항
- **사용자 경험(UX)**: 보안을 강조하되 잦은 다운로드 시 불편함을 주지 않아야 함. (하루 보지 않기 기능으로 해결)
- **적용 대상**: `ClassLogEditor`(학급일지) 및 `StudentCumulativeRecord`(누가기록)의 Excel/PDF 내보내기 버튼.

## 5. 성공 기준
- [ ] 내보내기 버튼 클릭 시 경고 모달이 정상적으로 표시됨.
- [ ] 모달에서 '확인' 클릭 시 정상적으로 다운로드가 진행됨.
- [ ] '오늘 하루 보지 않기' 체크 후 확인 시, 당일에는 다시 모달이 뜨지 않고 즉시 다운로드됨.
