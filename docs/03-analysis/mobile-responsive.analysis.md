## 3. 발견된 차이점 (Gaps) - 없음
1. **하단 네비게이션 개선**: 디자인 문서에는 구체적인 명세가 없었으나, 실제 구현 시 항목이 많아질 것에 대비하여 가로 스크롤(`overflow-x-auto`)을 추가하여 접근성을 높였습니다.
2. **출결 관리 UI 전환**: 좁은 모바일 화면에서 4개의 버튼이 가로로 배치될 경우 오클릭 위험이 있어, 모바일에서는 `select` 박스로 전환하는 지능형 UI를 적용했습니다.
