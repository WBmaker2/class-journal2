## 3. 발견된 차이점 (Gaps) - 없음
1. **Sticky 참조 패널**: 디자인 명세에는 없었으나, 긴 일지 작성 시에도 참조 정보를 계속 볼 수 있도록 `sticky top-0`를 적용하여 사용성을 높였습니다.
2. **데이터 시각화**: `AttendanceTracker`와 `StudentCumulativeRecord`의 우측 패널에 단순 텍스트가 아닌 요약 위젯과 아이콘을 배치하여 시각적 완성도를 높였습니다.
